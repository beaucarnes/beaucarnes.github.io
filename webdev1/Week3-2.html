##Bootstrap
###Bootstrap HTML5 & CSS framework that combines - Responsive Design - Grid Positioning - Mobile Templating - Javascript plugins
###About Bootstrap Bootstrap is a CSS framework that makes making halfway—decent—looking sites a snap! It was created by Twitter and was made open source a few years ago
###Where to get it http://www.getbootstrap.com > Getting Started Hit the "Download Bootstrap" button

How to use it

  1. Unzip the file and rename the folder "bootstrap"
  2. Make a copy of this new folder into the your project containing your html and css files
  3. Link to the bootstrap css in your index.html file
###Bootstrap Grids The best of CSS positioning and table layouts - 12 columns wide - Infinite rows - Responsive-ready out of the box - Uses only CSS
